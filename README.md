# Interactive Love Proposal 💖

A charming, interactive web application designed for a "Love Proposal" or "Valentine's Day" surprise. It features a playful UI where the "No" button becomes increasingly difficult to click, eventually moving randomly to ensure the user (hopefully) clicks "Yes"!

## ✨ Features
- **Dynamic Interaction**: Multiple pages of persuasion if the user clicks "No".
- **Evasive "No" Button**: On the final attempt, the "No" button moves randomly when the mouse approaches, adding a fun challenge.
- **Heart Animations**: Floating heart background animations for a romantic feel.
- **Background Music**: Synchronized audio that persists across page transitions using `sessionStorage`.
- **Cute GIFs**: Integration with Tenor GIFs to express emotions visually.
- **Responsive Design**: Works beautifully on both desktop and mobile devices.

## 🚀 Technologies Used
- **HTML5**: For the structure and sequence of the story.
- **CSS3**: For styling, animations, and the "Modern Nature/Romance" aesthetic.
- **JavaScript**: For the interactive logic, button movement, and audio synchronization.

## 📂 Project Structure
- `index.html`: The starting page with the big question.
- `no1.html`, `no2.html`, `no3.html`: Sequential "persuasion" pages.
- `yes.html`: The final celebration page once "Yes" is clicked.
- `script.js`: Contains the logic for the random moving button and audio handling.
- `style.css`: The styling for the entire application.
- `music.mp3`: The background track to set the mood.
